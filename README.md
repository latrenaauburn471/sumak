## 🗂️ Main Features

### 🧱 AST-First Design

sumak builds queries with an AST, which means it works with a structured form of the query before turning it into SQL. This helps keep query logic clear.

### 🔒 Type Safety

Type safety helps catch mistakes before your code runs. That can save time when building database queries.

### 🪶 Zero Dependencies

sumak does not rely on extra packages. This keeps the project small and easier to manage.

### 🧩 Hookable

Hooks let you connect extra logic at different points in the query process. This helps when you need custom behavior.

### ✂️ Tree-Shakeable

You can include only the parts you use. That can help keep your app smaller.

### 📝 Pure TypeScript

sumak is written in TypeScript only. That makes the codebase easier to read and keep in one style.
